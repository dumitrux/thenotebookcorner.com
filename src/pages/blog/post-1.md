## Development

Tools:

- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Tabler Icons](https://tablericons.com/)

```bash
npm create astro@latest
dir > .
tmpl > Include sample files
deps > Yes
ts > No
use > Strict

npm run dev
npm run build
npx astro add tailwind
```
